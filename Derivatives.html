How to use:

  • Enter a mathematical function in the input box (e.g., sin(x), x^2, etc.)
  • Drag the slider to move the point x along the curve
  • Observe the red tangent line - its slope is the value of the derivative at that point
  • The blue curve is the function f(x), the red dashed curve is the derivative f'(x)
